변수 : 변할 수 있는 수(Data)

[정수]
sbyte = -128~127 
byte = 0~255
short -32,768~32,767
int = -2,147,483,648~2,147,483,647
long = 8,223,372,036,854,775,808~9,223,372,036,854,775,807

[실수]
float : 32bit 부동 소수점(f)
double : 64bit 부동 소수점
decimal : 128bit 부동 소수점 (m) 

세미콜론(;) : 코드의 마지막을 알리는 기호

String은 char의 타입의 값을 모아 놓은 char 배열.

== : equals
&,&& : and
|, || : or
>, <, >=, <= : 비교 연산자
